## Welcome to Recur

Recur is a **privacy-first subscription tracker** that keeps all your data on your device. No cloud sync, no accounts, no tracking—just a beautiful, local-first app for managing your recurring expenses.

Built with React Native and Expo, Recur combines powerful features with a sleek, animated interface that respects your privacy.

## Tech Stack

Recur is built with modern React Native technologies:

| Technology | Purpose |
| --- | --- |
| **Expo** | React Native framework & tooling |
| **Expo Router** | File-based routing |
| **NativeWind** | Tailwind CSS for React Native |
| **React Native Reanimated** | Smooth, performant animations |
| **MMKV** | High-performance local storage |
| **Zod** | Runtime type validation |
| **Lucide Icons** | Beautiful icon set |
| **Draggable FlatList** | Drag-and-drop reordering |

<Note>
  Recur uses **Zod** for runtime validation to ensure data integrity when reading from local storage.
</Note>

## Data Models

Recur uses two primary data structures validated with Zod:

### Subscription

```typescript
{
  id: string;
  name: string;
  icon: string | null; // Emoji or null (falls back to initials)
  listId: string | null; // Associated list
  frequency: 'monthly' | 'yearly';
  amount: number;
  currency: Currency; // From CURRENCY_CODES
  startDate: Date;
}
```

### List

```typescript
{
  id: string;
  name: string;
}
```

<Warning>
  Lists are entirely optional. If a subscription's list is deleted, it becomes "Unassigned" and remains accessible via the Unassigned filter chip.
</Warning>
